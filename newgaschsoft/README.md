# Recursos compartidos de GaschSoft

La página pública se publica desde la raíz del repositorio (`/`).

- `assets/gaschsoft.css` contiene los estilos de la experiencia corporativa.
- `assets/gaschsoft.js` contiene la interacción, idioma y temas.
- `index.html` se conserva como una ruta compatible para `/newgaschsoft/`, pero su URL canónica es `https://gaschsoft.com/`.
- Los logos e iconos permanecen en `Logos/` para no duplicar recursos.

No ejecutar el despliegue legado de `landing-gaschsoft`: ese proyecto React se conserva únicamente como historial técnico y ya no alimenta la portada.
